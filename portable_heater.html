Portable Heater - Design Project(2017)

A mini hardware project. A heating element like nichrome or tungsten or even a heating pad was controlled using a microcontroller and sensors depending on the external temperature of the user's environment.

After a thorough study of microcontrollers and sensors, Arduine Lilypad was chosen as the appropriate one along with LM35 sensor.

  1. Hardware - Arduino Lilypad, LM35 sensor, Stainless Steel Threads, Power Supply Adapters, Arduino kit etc.
  2. IDE - Arduino IDE
  3. Programming Language - C++

The project was done as a part of academic requirement which stated that we had to create a project on any programming language of our choice as part of the subject 'Design'. The project was evaluated and marks were awarded. It was done in 2017.
